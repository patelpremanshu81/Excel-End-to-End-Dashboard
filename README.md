# Excel-End-to-End-Dashboard
An interactive Sales Dashboard built in Microsoft Excel to analyze and visualize business performance across different regions, categories, and time periods. This dashboard provides actionable insights into sales trends, customer behavior, and product performance.

Features
KPIs Overview
Total Sales
Sales Average
Total Transactions
Total Quantity Sold
Region-wise Analysis
Central, East, South, West filtering
Interactive slicers for dynamic insights
Ship Mode Analysis
Sales distribution by shipping types (First Class, Second Class, Standard Class, Same Day)
Time Series Analysis
Monthly quantity ordered trends
Top Performing Cities
Top 5 cities by sales
Category-wise Insights
Technology, Office Supplies, Furniture
Customer Segmentation
Consumer, Corporate, Home Office
Year-wise Filtering
2014–2017 interactive selection

Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Timeline Filters
Data Cleaning & Transformation
Dashboard Design & Data Visualization
